# Ethos Definitions

This page is the master ethos router for FloPo activity generation.

Do not use this page as the main source for writing full ethos adaptations. Use it to identify which ethos-specific skill doc to load for the current activity, then follow that skill doc for the actual generation.

## Purpose

Use this page when:
- the activity requires one or more ethos adaptations
- the generator needs to decide which ethos guidance is relevant
- an editor needs a quick comparison between ethos routes

This page should stay short. It is an index, not the full instruction set.

## Default position

[[The EYFS seven areas of learning & development]] is the default framework for all FloPo activities.

Ethos adaptations are optional alternative framings layered onto an EYFS-aligned activity. They must not be written as improvements, upgrades, or corrections to the default activity.

## Routing rule

When an activity includes an ethos adaptation:
1. Identify the ethos section being generated.
2. Open only the matching ethos-specific skill doc.
3. Use that skill doc to shape environment, materials, adult role, observation, and interaction.
4. Return to the main activity spec for section structure, tone, and validation rules.

Do not blend multiple ethos approaches inside one adaptation unless the task explicitly asks for comparison.

## Ethos skill docs

### Montessori

Use the [[Montessori Ethos Adaptation – Model Skill Reference]] when the adaptation should emphasize:
- independence
- ordered prepared environment
- real, purposeful materials
- precise adult modelling followed by withdrawal
- observation of concentration, repetition, and self-correction

### Forest School

Use the [[Forest School Research Ethos Adaptation — Model Skill Reference]] when the adaptation should emphasize:
- outdoor learning
- natural materials
- risk-managed challenge
- resilience and problem-solving in changing conditions
- relationship with weather, place, and repeated outdoor experience

### Reggio Emilia

Use the [[Reggio Emilia Ethos Adaptation — Model Skill Reference]] when the adaptation should emphasize:
- environment as the third teacher
- collaboration and co-construction
- open-ended expressive materials
- visible documentation
- adult dialogue that extends children's theories and shared meaning-making

### Steiner (Waldorf)

Use the [[Steiner (Waldorf) Ethos Adaptation — Model-Facing Skill Spec]] when the adaptation should emphasize:
- rhythm and flow
- imagination and storytelling
- sensory warmth and natural materials
- calm adult presence
- emotional tone, imitation, and whole-experience atmosphere

## Minimum output rule

Each ethos adaptation must still follow the main FloPo model-facing spec and [[Writing guide]]:
- practical adaptation, not philosophy summary
- concrete environmental or material change
- clear adult behaviour shift
- observation-and-response logic
- enough specificity that the activity would look and feel meaningfully different

If an ethos-specific skill doc conflicts with the core model spec, the model spec wins on structure and validation, and the ethos skill doc guides the content decisions inside that structure.
